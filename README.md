## Quick Start

Read the full Quartz documentation: https://quartz.jzhao.xyz/

### Useful Commands

- **Sync repository:**
	```sh
	npx quartz sync
	```
	(Replaces: `git add . && git commit -m "message" && git push`)

- **Build and serve the digital garden locally:**
	```sh
	npx quartz build --serve
	```
	(Opens at: http://localhost:8080)
